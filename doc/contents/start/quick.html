GoAhead Quick Start

  1. Download

    First download GoAhead to your system. You can download a binary package from https://embedthis.com/goahead/download.html.

  2. Building

    Compile GoAhead on your system by following the Building from Source Instructions.

  3. Read the doc

    There is a wealth of documentation to read. Try the User's Guide and Developer's Guide first.
