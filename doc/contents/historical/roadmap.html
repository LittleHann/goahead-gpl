Historial GoAhead Roadmap

This roadmap is no longer maintained. Please consult GitHub GoAhead Milestones for a current roadmap.

The GoAhead roadmap is created by listening to user input. As such, it is subject to change. This roadmap is not a commitment to product, but reflects our current thinking about future directions.
